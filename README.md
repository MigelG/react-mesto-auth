# React-mesto-auth
## Описание
Это одностраничное приложение, в котором пользователи могут размещать фотографии.
## Возможности:
* Регистрация пользователя
* Добавление и удаление карточек
* Добавление и удаление лайков у карточек
* Счетчик лайков
